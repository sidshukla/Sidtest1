Cloudreach Finsbury solution
============================

1. Compiling the code 
----------------------
--> Get the source code from GIT
--> Run mvn clean install under FinsburySolution directory to compile and package the code


2. Testing the code 
----------------------
--> Get the source code from GIT
--> Run mvn test to run the junit test cases

3. Running the code
------------------
--> Get the source code from GIT
--> Run mvn clean install under FinsburySolution directory
--> You will get a runnable jar file under target directory 
--> Run the jar file and you will get a command line menu
OR
--> Run the main class com.cloudreach.solution.main.FinsburySolution
